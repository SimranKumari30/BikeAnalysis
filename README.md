# Bike Sales Dashboard (Excel)

## Overview

This project presents an interactive Bike Sales Dashboard built in Microsoft Excel to analyze customer demographics and purchasing behavior. The dashboard enables quick, visual insights into how income, age, and commute distance relate to bike purchases.

The dashboard is designed for business users and analysts to explore trends dynamically using slicers without writing any code.

---
##Key Insights Visualized
1. Average Income per Purchase Compares average income by:
- Gender (Male vs Female)
- Purchase decision (Yes vs No)
Helps identify income trends among bike buyers.

2. Customer Age Brackets Breaks customers into:
- Adolescent
- Middle Age
- Old
Shows purchase behavior across age groups.

3. Customer Commute Distance Analyzes bike purchases based on commute distance:
- 0–1 miles
- 1–2 miles
- 2–5 miles
- 5–10 miles
- More than 10 miles
Highlights how commute length influences buying decisions.

---
## Interactive Filters (Slicers)

The dashboard includes slicers for easy filtering by:
- Marital Status (Married, Single)
- Education Level
- Region (Europe, North America, Pacific)
These filters update all charts simultaneously, enabling targeted analysis.
